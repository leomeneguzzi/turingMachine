# turingMachine

<br/>
For this program is necessary install Node.js: https://nodejs.org/en/download/
<br/>
The version i used now for Node is 8.9.1 and for npm is 5.5.1
<br/>
<br/>
To install necessary modules to run: "npm install"
<br/>
To start compiler of the typescript: "npm run tsc"
<br/>
To start the Electron application: "npm start"
<br/>
<br/>
OBS for the Turing Machine:
<br/>
*My empty value is "-";
<br/>
*For Right, Left and Stay use "R","L" and "S";
<br/>
*To separate the blocks os Transitions use "**";
